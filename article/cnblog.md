# 博客园之困

作为一个2011就开始在博客园写博客的老用户，可以说，博客园是我职业生涯中，非常重要的一部分，工作之余，我有相当一部分时间都用在了学习技术以及在博客园写技术博客。作为一个10年+的老用户，曾看着博客园日渐式微。甚至走向濒临关站的边缘。


## 博客园的优点、

刚工作之初，我几乎是同一时期注册了CSDN、51CTO、博客园、51testing 等技术社区的账号。为什么选择博客园，很重要的原因是 博客园 是一个纯粹写技术博客的社区，而且整个网站广告很少，界面比较干净清爽。另一个很重要原因是因为很是崇拜测试技术前辈， Jackei，有段时间拜读了他发表的所有文章。可是，他在 2012年就停更了，但，我想继承他的意志。继续分享测试技术。哈哈哈！

接下来，我要说博客园的问题了。

### 1. 基础服务没有做好

比如博客模板，一共就那么十几款吧！我已经好多年没换过模板了，甚至都找不到在哪里设置。这些模板一开始觉得还可以。后来随着 “轻博客” 的出现，简洁清新的界面顿时觉得 博客园的模板都过时了，何况这么年博客园似乎都没做大的改版。

首先，我知道 博客园 可以高度自定义样式，也有很多自定义的模板很好看。但是，并不是每个写技术博客的人都是懂前端 CSS 的。

其次，一个技术社区的成败肯定不是多提供几个好看的模板。我只是想说明 博客园 在技术服务方面，没有做好，或者说没有与时俱进。

另一个就是编辑器了。老的编辑器 TinyMCE 看着功能虽多，确实谈不上好用。后来提供了 Markdown 编辑器，我也是大概五六年前开始用 Markdown 写博客的。但是，博客园的 Markdown 体验很差，甚至Markdown 生成的排版也巨丑。​

如果体验过 飞书文档，语雀、或者是 51CTO 的编辑器，你就是知道博客园提供的编辑器体验是非常糟糕的。


### 2. 没有留住创作者

博客园 出现过许多大牛，他们的文章质量很高，短时间内就聚集了许多关注（粉丝）。可是，一段时间之后他们就在博客园停更了技术博客。 或者，因为工作太忙，或者他们自己搭建了博客。或者转战到新的平台。

博客园做了什么？ 什么也没做，无为而治，他来由他来，他走由他走！

我曾经被 阿里云、51cto 等多个平台要求入驻，送各种小礼品。要不是博客园已经留下我几百篇博客，我为啥不换个平台写？

殊不知，优秀的创作者的可贵，他们愿意持续在博客园创作，才能吸引更多的用户，有了更多的用户，才能有更多的流量，起码有更多广告变现吧。

### 3. 没有转型

博客园这么多年仍然是一个Web端技术社区。移动端来的时候，无动于衷，曾经有园友自己开发 博客园APP。甚至连公众号也没有。不是有头条，排行榜吗？那些高质量的文章，可以通过公众号的方式进行推送，这低成本增加​用户粘性的方式。博客园都懒得做。

### 4. 面对困境的骚操作

博客园这两年确实面临了诸多困难。评论功能被封，以及巨额罚款。巨量博客的审核工作是惊人的。博客园探索商业化本本来无可厚非，为了活下去嘛！可以....

1. 和某某护肤品合作，卖护肤品。

你是觉得博客园用户都是小鲜肉，每天上班都要美美的，每天下班各种护肤整一套？

2. 组织捐款

当一个平台，要靠捐款才能活下去，那这个平台距离死亡不远了。毕竟，每个人的情怀是有消费次数的。比如，我看到一个人很可怜，我可以娟一次钱给他，我不会也没有义务说，他今后的生活我包了，我每月固定打款给他吧！

前不久，天涯社区关闭。 网友自发的组织 7天 重启天涯，直播筹款活动。希望募集 300W，如果以天涯 巅峰时期的用户量，一人一分钱都够了。结果是筹集了20W 。直播内容挺有参考价值的。包括创始人 邢明 和 几个大佬 的对话内容。有兴趣可以找来看。

3. 会员救园
会员的权益，基本和没有差不多。如果要做会员，就提供一些真正的会员价值和服务​。否则就是"希望找个长期捐款的人"​。


## 如何脱困

博客园走到现在的困境，除了惋惜，其实我觉得也怪不得别人。当然，我也简单聊聊应对之方。浅薄建议，仅供参考。

### 周边合作

1. 博客园不是出过文化衫嘛？ 为啥不卖文化衫？ 哪怕你 20元成本得文化衫卖 50元，我也是愿意买两件得。反正夏天总少不了买几件衣服穿。这可比捐款博取同情强太多了。每年 1024 来个特别款，博客园成立 xx 周年，出个周年纪念款。只要打个博客园的 logo，质量太差。我是很愿意买的。 


2. 与机械键盘厂商合作，出联名款键盘。笔记本电脑，显示器等都可以做联名款。作为程序员这些都是我们的必须品吧！这些合作不比护肤品对口？
### 提升用户黏度

中长期发展：


1. 创作者奖励机制

B站粉丝 10W 都可以送奖牌。博客园得创作者也可以得到激励。比如 粉丝数达到多少，送奖牌。或者连续原创文章多少篇送奖牌。

2. 付费课程

园子挺多技术大牛，都有自己擅长的领域，完全可以参考 极客时间 联合园子的技术大牛推 付费课程。当然，上线这个功能还是有不少开发工作量的。

3. 长期规划

长期规划，围绕技术社区有很多方向可以做。
3.1 联合出版社 和 社区作者出版技术图书。可以分成。
3.2 举办技术大会，博客园技术嘉年华，可以有门票收入。
3.3 在线技术直播等。


## 最后

希望博客园能活下去，并转型成功，如果是现有路线继续走下去，我觉得关站只是时间问题。如果真的有一天关站了，我道是也不会觉得特别惋惜。他只是完成了自己的历史使命。就像很多古早的社区和游戏一样。我们除了怀念一下，还有新的平台满足我们需求。
